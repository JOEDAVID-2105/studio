# **App Name**: Joel David

## Core Features:

- Parallax Hero Section: Implement a scroll-driven parallax background animation using WebP frames, mapping scroll position to frame index with preload and opacity switching for a smooth cinematic effect.
- Intelligent Content Generation: Tool that dynamically tailors section content to highlight relevant skills and experiences based on the recruiter profile, utilizing the data in the CMS tool and understanding recruiter roles via AI. Coded under the name D'codes.
- Firebase Integration: Utilize Firebase Hosting, and other Firebase services, for a seamless user experience, real-time updates, and easy content management.
- Project Showcase: Display projects in a grid format with thumbnails, tech stack labels, and descriptions, each optimized for visual appeal and recruiter information.
- Loading Experience: Create a pre-loading screen with a progress bar and percentage indicator, revealing the hero section only after all assets are fully loaded.
- FAQ Accordion: Implement an accordion-style FAQ section with relevant questions and concise answers, ensuring a user-friendly and informative experience.
- Contact and Social Links: Provide multiple contact options, including email and GitHub, with clear call-to-action buttons and a consistent visual style, ensuring potential clients and employers can easily reach out.

## Style Guidelines:

- Primary color: Deep black (#000000) for a premium, cinematic feel.
- Background color: Very dark grey (#080808) for contrast with other page elements.
- Accent color: Vibrant orange (#F27405) for highlights, buttons, and loading indicators.
- Headline font: 'Space Grotesk' (sans-serif) for a bold, techy, modern look. Body font: 'Inter' (sans-serif) for readability and clean presentation.
- Use clean, minimalist icons with orange accents to represent links and actions, aligning with the cinematic theme.
- Single-page layout with a fixed navbar and rounded corners, designed to highlight each section, particularly the hero section, effectively.
- Smooth scroll-driven parallax animation and subtle hover states to enhance the user experience without distracting from the content.
